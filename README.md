# blog-activity

Your public blogging workspace - blog content, learning notes, and portfolio.

## 📁 Structure

- `content/blogs/` - All blog posts (published and drafts)
- `content/topics/` - Topic organization with learning notes
- `templates/` - Reusable blog templates
- `publishing/` - Platform-specific publishing configs
- `portfolio/` - Portfolio presentation
- `docs/` - Documentation

## 🚀 Quick Start

1. Write blog posts in `content/blogs/drafts/`
2. Organize by topic in `content/topics/`
3. Link to POCs in [agentic-ai-suite](https://github.com/yourusername/agentic-ai-suite)
4. Publish to platforms using configs in `publishing/`
